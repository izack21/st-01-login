# SUIT TEST 01 - Login

As váriaveis a seguir são obrigatórias para os testes: string username e string pass

O objetivo dos códigos presentes nesse repositório é validar o funcionamento do acesso via login na ferramenta.

# CT01
O objetivo do teste é validar tentativa de login sem dados de usuário.

# CT02
O objetivo do teste é validar o funcionamento do login de usuário.

# CT03
O objetivo do teste é validar o funcionamento do check-box para gravação dos dados de login do usuário.

# Pré-requisitos
Plug-ins Selenium WebDriver.
Plug-ins NUnit
